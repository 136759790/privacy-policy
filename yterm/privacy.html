1. 数据收集与存储

1.1 服务器连接信息：您在使用本应用过程中输入的Linux服务器IP地址、端口号、登录用户名（不含密码）等连接配置信息，仅用于实现远程服务器连接功能，全部存储在您的移动设备本地，不会上传至本应用服务器、第三方平台或云端存储服务。

1.2 敏感信息：本应用不会收集您的服务器登录密码、服务器内数据、操作记录等敏感信息；所有连接配置信息仅存储在您的手机本地，不会上传至任何服务器，也不会泄露给任何第三方。

1.3 设备基础信息：本应用仅在本地获取设备的网络状态（如Wi-Fi/移动网络）、系统版本信息，用于适配不同设备的连接兼容性，不会收集设备唯一标识符（如IMEI、OAID）、地理位置等隐私数据。

2.1 网络权限：用于建立手机与Linux服务器之间的网络连接，是应用核心功能必需权限，仅用于数据传输，不会监听、记录或上传网络传输内容。

2.2 存储权限：仅在您主动触发“导入/导出服务器配置”功能时申请，权限仅用于读取/写入您设备本地文件夹中的配置文件，不会访问、修改或上传设备内其他文件（如照片、文档）。

2.3 悬浮窗权限：可选权限，仅在您开启“后台连接提醒”功能时申请，用于在应用后台运行时显示连接状态提示，不会用于其他任何用途。

3.1 数据使用范围：本地存储的服务器连接配置信息仅用于您本人连接远程Linux服务器，应用开发者无法查看、获取或修改您的任何本地配置数据。

3.2 数据删除：您卸载本应用后，设备内存储的所有服务器连接配置信息将被自动彻底删除，且无法恢复；您也可在应用内手动删除单条/全部服务器配置。

3.3 数据备份：应用不提供自动备份功能，如需备份服务器配置，请手动导出至设备本地文件夹，备份数据由您自行保管。

4.1 免费且无广告：本应用为免费单机版工具，无任何付费功能，也不会展示第三方广告，不会因广告目的收集任何设备或个人信息。

4.2 无后台交互：本应用仅在您主动操作时建立与Linux服务器的连接，无后台运行的上传/下载进程，不会在您不知情的情况下传输任何数据。

Yterm - Privacy Policy

Effective Date: March 19, 2026

1. Data Collection and Storage

1.1 Server Connection Information: Linux server IP address, port number, login username (excluding password) and other connection configuration information you enter while using the App are only used to implement remote server connection functions. All data is stored locally on your mobile device and will not be uploaded to the App's servers, third-party platforms, or cloud storage services.

1.2 Sensitive Information: The App will not collect sensitive information such as your server login password, data within the server, or operation records; all connection configuration information is stored locally on your mobile device only, will not be uploaded to any server, and will not be disclosed to any third party.

1.3 Basic Device Information: The App only obtains the device's network status (such as Wi-Fi/mobile network) and system version information locally to adapt to connection compatibility of different devices, and will not collect privacy data such as unique device identifiers (e.g., IMEI, OAID) or geographic location.

2. Permission Usage Instructions

2.1 Network Permission: Used to establish a network connection between the mobile phone and the Linux server, which is a necessary permission for the core functions of the App. It is only used for data transmission and will not monitor, record or upload network transmission content.

2.2 Storage Permission: Applied only when you actively trigger the "Import/Export Server Configuration" function. This permission is only used to read/write configuration files in the local folder of your device, and will not access, modify or upload other files (such as photos, documents) in the device.

2.3 Floating Window Permission: Optional permission, applied only when you enable the "Background Connection Reminder" function, used to display connection status prompts when the App runs in the background, and will not be used for any other purposes.

2.4 Independent Permission Control: Except for the network permission which is necessary for core functions, all other permissions are optional. You can enable/disable them at any time in "Settings - Apps - Linux Remote Connect - Permissions" on your mobile phone. Disabling permissions only affects the use of corresponding auxiliary functions and does not affect the core server connection function.

3. Data Usage and Control Rights

3.1 Scope of Data Usage: Locally stored server connection configuration information is only used for you to connect to remote Linux servers. The App developer cannot view, access, or modify any of your local configuration data.

3.2 Data Deletion: All server connection configuration information stored on your device will be automatically and completely deleted and cannot be recovered after you uninstall the App; you can also manually delete single/all server configurations in the App.

3.3 Data Backup: The App does not provide automatic backup functionality. If you need to back up server configurations, please manually export them to the local folder of your device, and the backup data is kept by you.

4. App Nature Description

4.1 Free and Ad-free: This App is a free standalone tool with no paid functions and no third-party advertisements. It will not collect any device or personal information for advertising purposes.

4.2 No Background Interaction: The App only establishes a connection with the Linux server when you operate actively, with no background upload/download processes, and will not transmit any data without your knowledge.

5. Updates to This Privacy Policy

We may update this Privacy Policy in accordance with business adjustments or regulatory requirements. The updated policy will be announced in the App, and your continued use of the App constitutes your agreement to the updated policy.
